# Website 1 - Techno Mind Static Team Portfolio

## Purpose
This is the static website for the Internet Computing project. It presents Techno Mind team members, roles, skills, project idea, and final deployment links.

A bilingual Arabic/English team portfolio website presenting Techno Mind team members, role distribution, used technologies, and the required project deployment links.

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Responsive Design
- GitHub Pages
- Firebase Hosting

## Features
- Arabic and English language switcher
- Responsive design for mobile and desktop
- Team members section
- Skills section
- Project idea section
- Deployment links section
- Contact section
