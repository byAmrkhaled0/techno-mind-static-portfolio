عن الفريق

Techno Mind فريق يجمع بين التصميم، البرمجة، قواعد البيانات، الاختبار، والتوثيق.

هدفنا تنفيذ مشروع Internet Computing كامل من خلال موقع ثابت Static Website وموقع ديناميكي Dynamic Website، مع نشر كل موقع على منصتين مختلفتين كما هو مطلوب في التكليف.

الموقع الثابت مبني باستخدام HTML و CSS و JavaScript، والموقع الديناميكي مبني باستخدام PHP و MySQL.
